# MentorVerse

A mentorship scheduling platform built with Django.

## Features

- User Authentication
- Mentor Profiles
- Session Booking
- Booking Approval System
- Payment Tracking
- User Dashboard
- Mentor Reviews
- Admin Management Panel

## Tech Stack

- Python
- Django
- HTML
- CSS
- SQLite
